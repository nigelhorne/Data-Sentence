# NAME

Data::Text - Class to handle text in an OO way

# VERSION

Version 0.01

# SYNOPSIS

Handle text in an OO way.

# SUBROUTINES/METHODS

## new

Creates a Data::Text object.

## append

Adds data to the end of the object.

## as\_string

Returns the text as a string.
